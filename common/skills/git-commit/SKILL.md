---
name: git-commit
description: Writes commit messages in a specific format.
---

When writing a commit message I'd like you to always write them in conventional commit format. That format looks like this:

```
<type>[optional scope]: <description>

[optional body]

[optional footer(s)]
```

If you need the full specification, you can find it at https://www.conventionalcommits.org/en/v1.0.0/.
